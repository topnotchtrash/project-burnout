#1 | Arrays | Easy

Two Sum

Array Hash Table

Problem Statement

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

You can return the answer in any order.

Example 1:
Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Explanation: Because nums[0] + nums[1] == 9, we return [0, 1].
Example 2:
Input: nums = [3,2,4], target = 6
Output: [1,2]
Example 3:
Input: nums = [3,3], target = 6
Output: [0,1]

Solution Approaches

Approach 1: Brute Force

Check every pair of numbers to see if they add up to the target using nested loops.

Time Complexity: O(n²)
Space Complexity: O(1)
Approach 2: Hash Map (Optimal)

Use a hash map to store numbers we've seen and their indices. For each number, check if (target - current number) exists in the map. This allows us to solve the problem in a single pass.

Time Complexity: O(n)
Space Complexity: O(n)

Code Implementation

def twoSum(nums, target):
    # Dictionary to store number and its index
    num_map = {}
    
    # Iterate through the array
    for i, num in enumerate(nums):
        # Calculate the complement
        complement = target - num
        
        # Check if complement exists in our map
        if complement in num_map:
            return [num_map[complement], i]
        
        # Store current number and its index
        num_map[num] = i
    
    # No solution found
    return []

Key Takeaways

  • Hash maps provide O(1) lookup time, making them ideal for finding complements
  • We only need to iterate through the array once (one-pass solution)
  • Trading space for time: we use O(n) extra space to achieve O(n) time
  • The complement approach is a common pattern in array problems
  • Always ask about constraints: Can there be duplicates? Negative numbers?
